FPDS.gov Is Retiring

Searching contract data on FPDS.gov ends February 24, 2026.

After this date, contract data will only be available on SAM.gov.

Where Do You Need to Go?

Choose the option that applies to you:

Search Contract Data

If you previously searched contract awards in FPDS.gov, use SAM.gov going forward.

Starting February 24, search will only be available on SAM.gov.

Go to SAM.gov Search

Manage Contract Data (Federal Users)

Continue entering data through your agency's Contract Writing System (CWS).

Starting February 24, access contract data at sam.gov/contracting and select "Manage Contract Awards."

Go to Manage Contract Awards

Access Data via API

The FPDS Atom Feed will retire later in FY 2026.

Use the SAM.gov API instead.

Migration steps:

  1. Register for a SAM.gov API key
  2. Update your integration endpoint
  3. Validate JSON or CSV schema compatibility
Get API Key
